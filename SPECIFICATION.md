# 家庭AIリテラシー・ラボ 仕様書 【FINAL / FROZEN】

> **⚠️ 最終固定仕様**
> 本仕様書は 2025-12-26 をもって最終固定（FROZEN）されました。
> 今後、以下の例外を除き、一切の変更・機能追加・改善提案を禁止します。
> 1. 明確な不具合の修正
> 2. 法的・安全上の重大な問題への対処

## 1. プロジェクト概要
**名称**: 家庭AIリテラシー・ラボ
**目的**: 未成年を含む家庭において、AIを用いた「振り返り」「整理」「表現」を安全かつ静かに体験するための学習ポータル。
**主な機能**: AI処理は外部専門ツールに委譲し、本ポータルは「判断と導線」に特化する。

## 2. 不変原則（変更不可）
1. **成果測定なし**: 点数、進捗、ランキング、比較を一切行わない。
2. **商業色排除**: 収益、稼ぐ、バズ、拡散を想起させるUI・文言を一切使わない。
3. **生活優先**: 利用主体は保護者とし、学業と生活を最優先とする。
4. **導線特化**: AI処理は外部ツールを利用。
5. **自由意志**: やらなくても問題ない設計とする。

## 3. 画面構成
アプリケーションは以下の6画面のみで構成される。

1. **ログイン／認証** (`/login`)
   - シンプルな入場ボタンのみ。
   - 初回入場時に期間カウントを開始（LocalStorage使用）。
2. **ダッシュボード** (`/dashboard`)
   - ポータルのホーム画面。
   - 残り日数の表示。
   - 4キャラクターへのナビゲーション。
3. **教材ライブラリ** (`/library`)
   - 12時間相当の動画・テキスト教材（デモ表示）。
   - **「効率」「成功」等の文言を排除**。「ヒント」「暮らし」等の静かな表現を使用。
   - 再生済み・未再生の区別を行わない。
4. **実践ナビ：ひろげゾウ** (`/hirogezou`)
   - 文章表現・伝え方の学習ページ。
   - SNS等の外部投稿機能は持たない。
5. **保護者向け説明** (`/parents`)
   - 利用ルール、免責事項の明記。
   - **「労働ではない」「対価は発生しない」ことを明記**し、法的懸念を払拭。
6. **期限終了画面** (`/expired`)
   - 30日経過後に遷移。
   - 「終了」「年間利用権（準備中）」「次回案内待ち」の3択のみ。
   - **データの喪失等の不安を煽る表現を排除**。

## 5. UI/デザインルール
- **配色**: 落ち着いたパステルカラー、グラデーション。
- **スタイル**: グラスモーフィズム（Glassmorphism）を採用し、透明感と静けさを表現。
- **フォント**: Google Fonts 'Outfit' を使用。
- **ナビゲーション**:
  - 外部ツールは必ず `target="_blank"`。
  - 明示的な「戻る」ボタンは排除し、サイトタイトル等の自然な導線で戻る。
  - 「完了」「次へ」などの強制表現は禁止。

## 6. 技術スタック
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Global Variables, No Tailwind)
- **Font**: Next/Font (Google) - 'Outfit'

## 8. 運用ガイドライン (PROHIBITED ACTIONS)
**以下の行為は運用フェーズにおいて厳格に禁止される。**
1. **成果事例の作成**: ユーザーの作品を事例として紹介しない。
2. **成功ストーリーの演出**: 「これを使って変わった」等のストーリーを作らない。
3. **収益モデルの説明**: 将来的なマネタイズの可能性を含め、一切説明しない。
4. **SNS連携の示唆**: ハッシュタグの推奨や拡散を促さない。
5. **利用時間・回数の強調**: 「毎日やろう」「たくさん作ろう」と言わない。

## 9. テスト運用モード規定
- **定員**: 最大10名
- **期間**: 30日固定
- **禁止用語**: 「テスト」「実験」「フィードバック」「改善」
- **データ扱い**: 感想取得、アンケート、数値ログ取得を行わない。
